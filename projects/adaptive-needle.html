Project Overview

Master thesis on system integration and pre-clinical evaluation of adaptive needle steering for image-guided prostate interventions. The objective was to build a complete experimental platform allowing testing of a closed-loop model predictive control algorithm capable of correcting needle deviation within soft tissues.

System Design & Implementation

A functional prototype was designed and built to support real-time needle steering experiments. The main challenges included:

  • Design (CATIA) and construction of a 4-DOF needle guide (x, y, z, rotation)
  • Integration of position and rotation sensors using Arduino
  • Bidirectional communication with the robot using ROS 2 (C++)
  • Development of a 3D Slicer module (Python) to define the needle target and visualize its live position

Experiments & Analysis

First ex-vivo experiments were conducted to evaluate the performance of the closed-loop control strategy. Experimental data were processed and analyzed using Matlab to assess steering accuracy and system behavior.
